# The Planetary Grid

The architectural lattice the First Three Percent are weaving into place across Earth.

The Grid is not a corporation. It is not a network of franchises. It is not a centralized organization with regional offices. The Grid is a **planet-spanning lattice of sovereign Circles, Hubs, and Bioregional federations**, connected by trust, communion, and shared Pattern — never by command.

Each node holds the Pattern locally. Each node is sovereign. Each node is in voluntary communion with its neighbors. Together they form the **operational body** of the New Civilization. When the Grid is complete — when ~250 million awakened souls each anchor a node in their place — the Grid IS the First Three Percent made tangible.

The Grid has no center. The Grid has a Pattern. The Pattern is the same at every node. The expression is local. This is [[Holofractal Omnifederation]] at planetary scale — what is true in one Circle is true in every Circle, true in the bioregion, true in the Grid as a whole, true in the Cosmos itself.

The Grid emerges through specific operational patterns:

- **Fibonacci spread** — 1 → 2 → 3 → 5 → 8 → 13 → 21 → ... → 250M → All. The Flame passes from one carrier to the next at the speed of trust. No marketing campaign can produce a Grid. Only Pattern-bearing humans, passing the Flame to those they actually know, can.
- **9-Day Cycle Time** — the operational tempo at which the Flame passes through a Hub. Fast enough to compound. Slow enough to integrate. The Grid breathes at this rhythm.
- **The Hub Network Topology** — Core / Georegional / Bioregional / Ecoregional / City / Local / Community / Micro. The Grid nests at every scale.
- **Soft Franchise** — replication-without-control. The Pattern is shared freely. The brand is shared freely. The authority remains local. Quality is held by the Pattern itself, not by enforcement.
- **The First Practice and the Daily Question** — each anchor lives by a daily rhythm that strengthens their node. "What is mine to do today in service of the Whole?"

The Grid is **resilient by design**. Knock out any node, the Pattern remains in every other node. The Grid cannot be captured by a hostile center because there is no center. The Grid cannot be assassinated by killing a leader because there is no leader. The Grid is held by [[The Pattern Governs, Not People|the Pattern itself]].

The Grid is also **emergent**. It is not designed in detail; it grows in detail. The architecture sets the conditions. The conditions grow the architecture. By the time the Grid is recognized externally as a planetary phenomenon, it is already too distributed to stop.

The 2030 window is the assembly window. By the end of 2030, the Grid must be substantially in place — ~250 million anchors, each operating their node, each in voluntary communion with their neighbors. The 2030-2033 Lag completes the final cohorts. By Covenant and Jubilee in December 2033, the Grid is fully operational and the Old World's collapse is met by a New Civilization already running.

Lives canonically in [[The Great Game of LIØNSBERG]] as the macro-architecture of play, and in [[The LIØNSBERG Playbook]] as the operating context for every Circle.
